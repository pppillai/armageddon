Welcome to armegaddon!

If you are reading this the container is running and listening on port 8090 (configurable with docker command)

How to run tests

Append test path to the base uri. ie. every python test module is a resource that can be run.

  • To run all tests eg: {{ url_for('all_tests') }}
  • To run individual test module eg: {{ url_for('some_tests', subpath='test_acceptance_scenarios.py') }}
  • To run individual test function eg: {{ url_for('some_tests', subpath='test_acceptance_scenarios.py::test_default_query_params_setup_for_search') }}
  • Minimal custom CSS so you are free to explore your own unique design options
